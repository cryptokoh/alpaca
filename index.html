The Stack

Three layers. One pipeline.

Each layer handles what it does best — intelligence, translation, and fabrication.

🤖

Claude Code

AI reasoning engine that understands your intent, generates optimized G-code parameters, troubleshoots print issues, and iterates on designs through natural language.

INTELLIGENCE LAYER
🔌

OpenClaw

Open-source bridge between AI commands and printer firmware. Translates high-level instructions into precise machine operations with real-time feedback loops.

TRANSLATION LAYER
⚙️

Flashforge AD5M Pro

High-speed CoreXY printer running Klipper firmware. 600mm/s capable with auto-leveling, filament detection, and network control built in.

FABRICATION LAYER

// Alpaca3D pipeline
const idea = "phone stand with cable management";

const design  = await claude.generate(idea);      // AI designs it
const gcode   = await openclaw.translate(design); // OpenClaw slices it
const result  = await ad5m.print(gcode);          // AD5M Pro prints it

console.log(result.status); // "complete" ✓
